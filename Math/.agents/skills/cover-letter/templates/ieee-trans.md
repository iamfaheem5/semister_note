# IEEE Transactions Cover Letter

> Snapshot for IEEE Transactions titles (TPAMI, TIP, TIT, TKDE, TASE, TSP, TVT, ...). IEEE Transactions editors value methodological clarity and rigorous comparison more than broader-impact framing.

## Tone

- Methodologically precise. Name techniques, baselines, and evaluation protocols.
- Conference-paper extensions are common — disclose them explicitly with a percentage of new content.

## Opening

- First sentence states the methodological advance and the problem class it addresses.
- Reference the manuscript type (regular paper / brief / survey) and target Transactions title.

## Contribution Framing

- Three to four bulleted or numbered contributions, each one sentence.
- For each contribution, include the type (algorithmic, theoretical, empirical) and a quantitative anchor when applicable.
- If this is an extension of a conference paper, name the prior venue and state the percentage of new content.

## Journal Fit

- Two to three sentences connecting the work to the Transactions' scope.
- Cite one or two recent papers from the target Transactions that this work extends.

## Declarations (Required for IEEE)

- Originality: "The manuscript has not been published elsewhere and is not under concurrent consideration."
- Conference extension disclosure: if extending a prior conference paper, state the conference and the percentage of new material (IEEE typically expects ≥30% new content).
- Competing interests.

## Length

- Hard ceiling: 500 words. IEEE allows more length than top journals because methodological context often requires it.

## Common Desk-Reject Triggers

- Conference extension without disclosure.
- Insufficient comparison with state-of-the-art (IEEE reviewers expect strong baselines).
- Missing experimental detail in the contribution framing.
