## 2018 Examination

**Course Title:** Discrete Math (Math-203)  
**Full Marks:** 70 | **Time:** 3.0 hrs  
**Instructions:** Answer Any Five Questions

### Question 1
(a) Define Set with example.  
(b) Explain Venn Diagram for Union and Intersection operations where, A and B are two sets.  
(c) Write down the Laws of set theory.  
(d) Prove that |A₁ ∪ A₂| = |A₁| + |A₂| - |A₁ ∩ A₂|, where A₁ and A₂ be sets with cardinalities |A₁| and |A₂|.

### Question 2
(a) What is the Recursion? Obtain the recurrence relation for G(k) = 2.4ᵏ - 5(-3)ᵏ.  
(b) What is a Propositional Calculus? Find the direct and indirect proof of p → (q → s).  
(c) Draw the Block Diagram with Truth Table for two input Logic Gate AND, OR and NOT.

### Question 3
(a) What is an Incidence Matrix? Construct the Incidence matrix for the following graphs G₁ and G₂.  
(b) What is a Pseudograph? Write down the Degree of a vertex for a Uni-directed graph G₁ and Directed graph G₂ respectively from the following figure, where the degree of a vertex denoted by d(Vᵢ).  
(c) Draw the digraph from the following adjacency matrix (Aᴳ).

### Question 4
(a) Explain Euler paths and Euler circuits in Graphs with example.  
(b) What is meant by Functions and Inverse Functions with proper example?  
(c) Let, R be the relation with ordered pairs (Abdul, 22), (Kalam, 24), (Azad, 21), (Desire, 22), (Eddie, 24), and (Felicia, 22). Here each pair consists of a graduate student and this student's age. What is the function that this relation determines?  
(d) What are the differences among Injections, Surjections and Bijections?  
(e) Let, f and g be the functions from the set of integers to the set of integers defined by f(x) = 6x + 3 and g(x) = 5x + 2. What is the composition of f and g? What is the composition of g and f?

### Question 5
(a) Let, f be the function from {a, b, c} to {1, 2, 3} such that f(a) = 2, f(b) = 3, and f(c) = 1. Is f invertible, and if it is, what is its inverse?  
(b) Define binary relation with example.  
(c) Let, set A = {1, 2, 3, 4}. Which ordered pairs are in the relation R = {(a, b) | a divides b}?  
(d) Explain reflexive relation and Equivalence Relation with suitable example.  
(e) What is the composite of the relations R and S where R is the relation from {1,2,3} to {1,2,3,4} and S is the relation from {1,2,3,4} to {0,1,2}.  
(f) What do you mean by Graphs of relation?

### Question 6
(a) Let, A = {1,2,3} and R be the relation, R = {(1,2), (1,3), (2,2), (2,3), (3,3)}. Draw the digraph by using this relation.  
(b) Draw the Hasse diagram representing the partial ordering {(a,b) | a divides b} on set {1, 2, 3, 4, 6, 8, 12}.  
(c) Let, set A = {1,2,3} and the relation r from A into A as r = {(1,2),(1,3),(2,1),(1,1),(2,3)}. Draw the relation matrix.  
(d) Write short notes on Relational Database.  
(e) State and proof the Principle of Inclusion and Exclusion.

### Question 7
(a) Define semi groups and Monoids with example.  
(b) Write down the properties of a Group G, where G is a non empty set together with a binary operation *.  
(c) What is a Subring? Write down the relationship between a Communicative Ring and Zero Divisors.  
(d) Prove that for any two ideals A and B of a Ring R, product AB is an ideal of R.

**Instructions:** Answer any five questions

### Question 1

(d) In a class of 80 students, 50 students know English, 55 know French and 46 know German language. 37 students know English and French, 28 know French and German, 7 students know none of the languages. How many students know all the 3 languages?

(d) Simplify the proposition using laws of algebra of propositions ¬(p ∨ q) ∨ ¬(¬p ∧ q).  
(e) Determine which rule of inference is used in the following arguments.

### Question 4
(a) Define reflexive, symmetric and transitive relation.

(d) Let A = {1,2,3,4,5,6} and let R be the relation on the set A; R = {(x, y) | x divides y}. Write R as a set of ordered pairs.

### Question 5
(a) Define Injective, Surjective and Bijective function with example.  
(b) Give an explicit formula for a function from the set of natural numbers (N) to the set of natural numbers (N) that is: i) one-to-one, but not onto. ii) onto but not one-to-one. iii) one-to-one and onto. iv) neither one-to-one nor onto.  
(c) f, g, h are functions on X = {1, 2, 3} as given. Compute i) f ∘ g ∘ h ii) f ∘ h ∘ g.
