## Study Notes
- **Matrix questions are never standalone** — they are always combined with either Relations or Graph Theory.
- Know how to:
  - Convert a graph to adjacency/incidence matrix and vice versa.
  - Convert a relation to matrix form and vice versa.
  - Compute join (element-wise OR) and meet (element-wise AND) for zero-one matrices.
  - Compute R² using Boolean matrix multiplication (or standard matrix multiplication for relations).
- Adjacency matrix for directed/undirected graphs: A[i][j] = number of edges from vertex i to j.
- Incidence matrix: rows = vertices, columns = edges.
