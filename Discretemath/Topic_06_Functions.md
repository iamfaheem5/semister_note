## Core Concepts Tested
- Injective (one-to-one), Surjective (onto), Bijective functions — definitions with examples
- Composition of functions (f∘g, g∘f, f∘g∘h)
- Inverse functions
- Floor and Ceiling functions
- Domain and Range
- Explicit formulas for functions with specific properties
- One-to-one and onto function with given formulas

## Study Notes
- **Composition of functions** is tested very frequently, especially with f(x) = 2x+3 / 3x+2 type formulas and with discrete set functions on X={1,2,3}.
- **Injective/Surjective/Bijective definitions with examples** is a standard theoretical question.
- **Floor/Ceiling** has appeared in 3 consecutive recent years (2021, 2023, 2024). Know: ⌊x⌋ is greatest integer ≤ x, ⌈x⌉ is smallest integer ≥ x.
- **Domain and range** appeared in 2024 — know basic real-analysis style domain/range for simple functions.
