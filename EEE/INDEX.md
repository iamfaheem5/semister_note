## ⚠️ Extraction Notes

1. **Image-based PDFs**: All question papers are scanned/image-based (CamScanner). Direct text extraction returned 0 characters for most files. Vision-based transcription was performed by rendering each PDF page to PNG at 2× zoom (~144 DPI) and manual reading.
2. **Text-layer exception**: The 2021 Final paper had a partial text layer that allowed some direct extraction, though it contained OCR artifacts (e.g., `delermine` → `determine`, `1hevenin` → `Thevenin`). These were corrected during topic-note compilation.
3. **Marks uncertainty**: The 2021 paper text extraction did not preserve all marks allocations clearly. Marks for 2022–2024 were read directly from the images.
4. **Theory solutions**: `EEE-1103_Semester_Final_Questions_21-23_Theory.pdf` contains handwritten **answers/solutions**, not questions. It was excluded from topic-frequency counting but retained in the source index for reference.
5. **Solution manuals**: The two `Solve_20xx` files are solution sets, not question papers, and were excluded from analysis.
