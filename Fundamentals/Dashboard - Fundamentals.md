## Exam Pattern Insights

### Incourse Exams (20 Marks, 1 Hour)
- **4 broad questions** covering major theory topics
- Common themes: Software/OS basics, Memory, Programming fundamentals, Networks, I/O devices

### Semester Final (70 Marks, 3 Hours)
- **7 questions with sub-parts (a, b, c, d)**
- Answer any 5 out of 7
- Mix of theory, definitions, comparisons, numerical calculations, and C programs
- **Numerical problems** appear consistently: Number system conversions, disk access time calculations
- **C programming** is a guaranteed section (basic structure, algorithms, flowcharts, conditionals)

---

## High-Yield Topics (Must Prepare)
1. **C Programming** - Guaranteed in finals. Structure of C, variables, data types, flowcharts, simple programs.
2. **Memory Systems** - Appears every single year without fail. Know RAM types, cache, HDD/SSD working, disk calculations.
3. **Networks** - Topologies, LAN/MAN/WAN, network devices (hub/switch/router), IP/MAC.
4. **Computer Organization** - Digital computer block diagram, components, data processing cycle, ports/interfaces/buses.
5. **OS Basics** - Multitasking, booting, file systems, types of OS.
